We need to install few packages to convert text to speech
-------------------------------------------------------------

1- "pyttsx3" is a text-to-speech conversion library in Python

    pip install pyttsx3

--------------------------------------------------------------

2- "PyPDF2" is to extract PDF data into text

    pip install PyPDF2

---------------------------------------------------------------

3- "docx" is to extarct MS word data into text 

    pip install python-docx
 
 --------------------------------------------------------------
 
4 - "pptx" is to extact Power point slied data into text

    pip install python-pptx
    
---------------------------------------------------------------

We can extact word and other file data too, just need to install specific module
